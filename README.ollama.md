# Mistral Trismegistus 7B

> "Transcendence is All You Need!"

The first powerful open-source LLM dedicated to **esoteric, occult, and spiritual** knowledge.

## Overview

- **Base Model**: [mistralai/Mistral-7B-v0.1](https://huggingface.co/mistralai/Mistral-7B-v0.1)
- **Parameters**: 7B
- **Quantization**: Q6_K (5.9 GB)
- **License**: Apache 2.0
- **Original**: [teknium/Mistral-Trismegistus-7B](https://huggingface.co/teknium/Mistral-Trismegistus-7B)

## Capabilities

- Esoteric and occult knowledge
- Spiritual guidance and philosophy
- Hermetic traditions
- Ritual and meditation scripts
- Symbolic interpretation
- Oracle and divination responses

## Training

- **Dataset**: 100% synthetic, GPT-4 generated
- **Size**: ~10,000 high-quality examples
- **Focus**: Deep, diverse esoteric and spiritual tasks

## Usage

```bash
ollama run arthurjean/mistral-trismegistus:7b-q6_K
```

### Prompt Format

```
USER: <your question>
ASSISTANT:
```

Or with system message:

```
<system message>
USER: <your question>
ASSISTANT:
```

## About This Version

This quantized version (Q6_K) is optimized for the [Quantum Oracle](https://github.com/your-repo) project, configured with a French mystical system prompt for oracle-style responses.
